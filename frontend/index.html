About

Secret keeper is a free forever service which is built to help of solving a problem of transferring secrets (like a credentials: user, passwords, etc...) by email or chat. Yet another thing is to use this service as a second factor. In case of losing one part of data, the second still unknown.

Secret keeper is an open source project, you can always check the source code on Github. It also provides a REST API service to automate the process of generating a one-time secrets links.

  • Create a new secret
  • Send a generated one-time link to recipient
  • Follow the link to view the secret
